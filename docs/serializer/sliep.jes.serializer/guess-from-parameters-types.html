guessFromParametersTypes

fun <M : Executable> guessFromParametersTypes(clazzName: String, members: Array<out M>, name: String?, params: Array<out KClass<*>?>): M

Makes possible to find a compatible method having only it's parameter types (or sub types) and the name

Parameters

clazzName - only for debug: the name of declaring class

members - all methods in that class

name - of the method

params - method arguments types (null types are jolly, this can lead to ambiguity problems)

Exceptions

NoSuchMethodException - if a matching method is not found.

Author
sliep

Return
the matching method
