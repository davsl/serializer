callGetter

fun <R> Any.callGetter(fieldName: String): R

Call kotlin (or java) property getters of a receiver object through reflection

No matter if it's not accessible

If the property name is 'foo', 'getFoo()' method will be called (if exists)

Parameters

R - Type of return value

fieldName - of the property

Exceptions

NoSuchMethodException - if a matching method is not found.

Author
sliep

Receiver
the instance having the magic property

Return
the value of the wanted property
