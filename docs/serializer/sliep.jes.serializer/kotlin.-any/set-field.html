fun Any.setField(name: String, value: Any?, inParent: Boolean = true): Unit

Set the property value of a receiver object through reflection

No matter if it's not accessible or final

Parameters

name - of the property

value - to be set

inParent - tell the function to search the field not only in the declaring class, but even in it's superclasses. default is true

Exceptions

NoSuchFieldException - if a field with the specified name is not found.

Author
sliep

Receiver
the instance having the magic property or the property declaring class to fetch a static value
