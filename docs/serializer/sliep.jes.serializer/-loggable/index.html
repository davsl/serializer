Loggable

interface Loggable

Logging utility

  1. Implement Loggable
  2. Define a static Boolean named 'LOG' (e.g. LOG = BuildConfig.DEBUG)
  3. (optional) Define an instance field Int named 'depth'
  4. in your Loggable class just call log
