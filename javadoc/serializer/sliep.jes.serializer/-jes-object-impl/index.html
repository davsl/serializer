JesObjectImpl

interface JesObjectImpl<T : Any> : JesObject

Let your class implement JesObjectImpl instead of JesObject to control it's serialization Implementing the method toJson you convert the instance into a value of type T A JesObjectImpl instance must provide a constructor having JesConstructor<T> as only parameter to be deserialized as well

Parameters

T - type of Json field in which the object will be serialized
